# **App Name**: Rozua Point & Legacy

## Core Features:

- Welcome Splash Screen: Display an elegant, dark-mode splash screen with smooth transitions upon loading the website.
- Elegant Menu Navigation: Implement a full-screen menu with three editorial-style cards for Vinos, Automóviles, and Inmuebles.
- Vinos Section: Curated selection of legacy vineyards and limited-edition wines.
- Automóviles Section: Showcase iconic models, collectible pieces, and limited-edition supercars.
- Inmuebles Section: Feature properties with character: restored mansions, historic estates, and discreet residences.
- Dark/Light mode persistance: Offer a toggle for automatic light and dark themes with user preference persistance

## Style Guidelines:

- Background: Soft black (#0E0D0C) or Ivory (#F5EFE6) for elegance and readability.
- Accent: Old gold (#C2A878) for button backgrounds and borders, providing a luxurious touch.
- Font: 'Cormorant Garamond', a serif font for a premium, old-money feel in headings and key text elements. Note: currently only Google Fonts are supported.
- Font: 'Alegreya', a serif font, for subtitles and descriptions for an elegant and refined look. Note: currently only Google Fonts are supported.
- Editorial-style card layout for the menu sections, mimicking a luxury magazine or auction catalog.
- Subtle animations using Framer Motion for button interactions (scale + fade), screen transitions (fade to black), and menu cards (staggered reveal).
- Micro-interactions: Menu card zoom 1.05 on hover, opacity +10%, subtle golden border transition, title reveal.
